# Codex - TikTok Channel Content System

This repository contains the working brand, voice, and content system for the Hebrew TikTok/Reels/X channel.

It is not a portfolio website.
It is the creative operating system for the channel: voice rules, content pillars, reusable formats, writing exercises, production guidance, and a starting content bank.

## What Is This Project?

The channel is built around one core idea:

> קומדיה של ראיות קטנות נגד דברים שכולם החליטו שהם נורמליים.

The voice does not explain why the world is broken.
It shows where it gets stuck.

## Project Files

- [Brand Bible](docs/01-brand-bible.md) - the core identity, tone, point of view, and writing rules.
- [Content Bank](docs/02-content-bank.md) - ready-to-use post ideas, short lines, and Reel concepts.
- [Creation System](docs/03-creation-system.md) - questions, formulas, and exercises for generating new ideas.
- [Production Guide](docs/04-production-guide.md) - how to perform, film, caption, test, and organize content.

## Core Rule

Do not write a thought.
Write evidence.

Bad:

> אנשים מתישים אותי

Good:

> יש אנשים שנכנסים לחדר וישר מרגיש כאילו נפתחה קבוצת וואטסאפ שלא ביקשת

## Current Working Definition

זו דמות שמסתכלת על החיים היומיומיים כאילו הם מערכת שלא עברה בדיקות, ומביאה ראיות קטנות: הודעה, בוקר, תור, מעלית, שיחת טלפון, עבודה, אדם שעומד במקום הלא נכון.

היא לא מתלוננת.
היא מתעדת תקלות.

## Next Workflow

1. Use the content bank to pick 10 strongest pieces.
2. Mark each one as:
   - feels exactly right
   - close but needs sharpening
   - not the voice
3. Use the creation system to generate 10 more from real situations.
4. Film 5 short tests.
5. Refine the voice based on what feels natural in performance.
